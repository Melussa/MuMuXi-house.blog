使用说明

1、构建

下载confirm.js并引用,同时将confirm.css引入你需要使用该插件的页面或者是全局

2、调用方式

调用方式如下：

	plugin.confirm({
	        title: '查看提示', /* 按需填写 */
	        message: '是否关闭？',/* 按需填写 */
	        buttons: [
	            {
	                text: '取消',
	                callback: () => {/* 此处回调按需书写即可 */
	                }
	            },
	            {
	                text: '确定',
	                callback: () => {/* 此处回调按需书写即可 */
	                   
	                }
	            }
	        ]
	    });

3、优点

a、可根据自身实际使用需求，修改CSS 或者 JS 文件

b、调用API简单易上手，切容易理解
